MediaPlayer.js

Un Media Player muy inteligente

Tiene 2 plugins muy inteligentes para reproducir automaticamente el video, y para pausarlo cuando ocupa menos del 25% de la pantalla para así ahorrar Internet.

Comentarios

  • Estos comentarios son útiles para demostrar el AutoPause cuando el video ocupa menos del 25% de la pantalla, son solo funcionales.
  • Vale aclarar que este proyecto no uscó un lindo diseño, ni se centró en eso. Lo que prueba este proyecto son las habilidades de Javascript, demostradas en los plugins
